Git (Version Control)

Menard always use Git Version Control to his projects because it allows him to easily jump from version to version, and it makes debugging and introducing new code easier and more safe.

He usually use separate branches everytime he's touching the old code, when debugging, or introducing new feature, so that he can ensure that he has a back-up cleaner and safer code in case something's gone wrong or to prevent introduction of breaking changes.

He also use GitHub to host his remote repositories so that he has a backup of his code.

React Python Scraper

Live Front End Back End
Description

A Website showing scrapped public data (by Python) about the latest Reddit posts in Crypto-currencies.

The Front-end is just a Simple React/Material UI Page, while the Python Backend consists of Web Scraper Bot to scrape Reddit Data. The scraper is run by a Shell Script which also manages the updating of data. The shell script runs every 2 hours by Linux CRON Job. The scraped data is served by the Python-Flask API (in which the React front-end connects into).

NOTE: As of now, the Webscraper bot is paused by Menard, thus, the data presented should be outdated and irrelevant

Tech Stack

Utils

    2. Shell File: Execution of Web Scraping, and Updating Data
